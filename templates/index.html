Open Source NPS System

A modern, self-hosted Net Promoter Score system with AI-powered feedback analysis, vector embeddings, and email integration.

Get Started View on GitHub

Key Features

Vector Embeddings

Store feedback as vector embeddings for advanced analysis and similarity search.

AI-Powered Analysis

Automatic feedback summarization using OpenAI's GPT models.

Email Integration

Built-in email support with Resend for production and Mailpit for development.

Modern Stack

Built with Express.js, Alpine.js, Tailwind CSS, and Supabase.

Security First

Rate limiting, input validation, and comprehensive security features built-in.

Developer Friendly

Comprehensive documentation, API endpoints, and local development tools.
